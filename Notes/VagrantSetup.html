- [Download and install VirtualBox 4.3.36](https://www.virtualbox.org/wiki/Download_Old_Builds_4_3) - [Download and install KiTTY](http://www.9bis.net/kitty/?page=Download) - [Download and install Vagrant](https://www.vagrantup.com/downloads.html) - [Find your desired Vagrant Box](https://atlas.hashicorp.com/boxes/search) - Using `ubuntu/trusty64` as an example + `cd ~` + `mkdir OS` + `cd OS` + `mkdir Main` + `cd Main` + `vagrant init ubuntu/trusty64; vagrant up --provider virtualbox` - SSH in with KiTTY, and you're good to go!
