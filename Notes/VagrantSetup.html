- [Download and install VirtualBox 4.3.36](https://www.virtualbox.org/wiki/Download_Old_Builds_4_3) - [Download and install KiTTY](http://www.9bis.net/kitty/?page=Download) - [Download and install Vagrant](https://www.vagrantup.com/downloads.html) - [Find your desired Vagrant Box](https://atlas.hashicorp.com/boxes/search) - Using `ubuntu/trusty64` as an example + `cd ~` + `mkdir OS` + `cd OS` + `mkdir Main` + `cd Main` + `vagrant init ubuntu/trusty64; vagrant up --provider virtualbox` * If you get errors about how Vagrant isn't installed, use [a Windows path editor](https://github.com/rix0rrr/WindowsPathEditor) - SSH in with KiTTY, and you're good to go! --- **Optional** - Perhaps you'd like to add a basic GUI to your headless virtual machine. Just invoke `sudo apt-get install ubuntu-desktop --no-install-recommends` in the virtual machine
